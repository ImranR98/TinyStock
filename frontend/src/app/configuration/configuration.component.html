Configuration
This is where TinyStock data is stored TinyStock needs the path to a folder where application data can be stored. You may use an empty folder or an existing TinyStock data folder.
Minimum 12 characters - UNRECOVERABLE Remember Password
TinyStock data must be password protected. If you're using an existing TinyStock data folder, enter its existing password. If you're setting the password for the first time, make a note of it in a safe place, as there is NO WAY to recover it. Passwords must be at least 12 characters long. If you choose to 'remember' the password, you won't have to enter it here on every app launch but it will be available to anyone with access to this computer. Leave this empty if unsure
